# Auto-GPT Notion Plugin
> Under rapid development

Functionalities:
- Read Notion pages
- Write to Notion pages

Applications:
- Automatically collects information from the web and archives it to Notion.
- Notion as a knowledge base, chat with it on Discord/Slack.
